Session 4 — Water Electrolysis and Gas Collection

Session 4

Hook: Split water, collect gases, and see a 2:1 stoichiometric ratio.

Learning outcomes

  • Build a gas collection apparatus by water displacement
  • Run water electrolysis safely with a non-chloride electrolyte
  • Measure H₂ and O₂ volumes and compare to 2:1 ratio
  • Connect electrolysis to hydrogen as an energy storage concept

Session plan (90 min)

Time Activity Reference
0–10 min Hook: inverted water-filled tubes — which gas appears faster? lecture.md
10–25 min Water splitting and stoichiometry lecture.md
25–40 min Build gas collection setup experiment.md — Build
40–65 min Run electrolysis; record volumes over time experiment.md — Run
65–75 min Instructor-only H₂ pop test (optional) experiment.md — Pop test
75–85 min Energy storage discussion lecture.md
85–90 min Preview Session 5 — reversing the reaction lecture.md

Key reactions

Overall: 2H₂O(l) → 2H₂(g) + O₂(g)

Cathode (−): 2H₂O + 2e⁻ → H₂ + 2OH⁻
Anode (+):   2H₂O → O₂ + 4H⁺ + 4e⁻   (acidic media)
         or 4OH⁻ → O₂ + 2H₂O + 4e⁻   (basic media)
