Session 5 — Preparation

Session 5

Before class (instructor)

  • [ ] Confirm derusting cells were started Session 4 end-of-class (see Session 4 prep)
  • [ ] Photograph nails before treatment if not done Thursday
  • [ ] Prepare saltwater for galvanic demos (label "corrosion only — not for gas electrolysis")
  • [ ] Clean metal strips; sand if oxidized
  • [ ] Print final challenge worksheets

Session 4 handoff (critical)

Derusting cells must be running before students leave Session 4:

Item Detail
Electrolyte Washing soda preferred; baking soda OK — not NaCl
Cathode (−) Rusty iron
Anode (+) Sacrificial steel or graphite
Control Rusty nail in plain water, no power
Power 9 V battery; secure clips; label each cell

Optional multi-day corrosion setup

Start Monday or Tuesday for best rust by Friday:

Jar Setup
B Iron nail wired/touching copper in saltwater
C Iron nail wired/touching zinc in saltwater
A Iron alone (control)

Store open jars where students can observe daily changes.

Day-of timeline

When Task
0 min Reveal derusted nails + corrosion jars (Part C)
25 min Metal-pair galvanic cells (Part A)
45 min Corrosion jar observations (Part B)
70 min Week comparison table (Part D)
80 min Final challenge; collect worksheets

Rust removal troubleshooting

Issue Fix
Little visible change overnight Fresher rust; higher soda concentration; check connections
Anode heavily corroded Expected — sacrificial behavior
Cloudy solution Normal; rinse nail before comparing
Chlorine smell Stop — salt was used; replace electrolyte
