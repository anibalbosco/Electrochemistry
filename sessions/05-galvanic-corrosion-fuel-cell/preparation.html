Session 5 — Preparation

Session 5

Before class (instructor)

  • [ ] Test PEM kit full cycle: electrolyzer mode → fuel cell mode → LED/fan
  • [ ] Note electrolysis time required for usable gas storage
  • [ ] Prepare saltwater for galvanic demos (label "corrosion only — not for gas electrolysis")
  • [ ] Clean metal strips; sand if oxidized
  • [ ] Print final challenge worksheets

Day-of timeline

When Task
−30 min Start PEM kit electrolysis if gas needed early
0 min Display corrosion jars + hook materials
55 min Switch kit to fuel cell mode (or pre-charged from before class)
80 min Collect worksheets; photo of week artifacts optional

Fuel cell troubleshooting

Issue Fix
No power in fuel cell mode Run electrolyzer longer; check water level
Weak fan/LED Verify gas reservoirs; connections
Kit leaks Tighten per manual; dry contacts
