# Session 5 — Galvanic Corrosion and Fuel Cell Demo

**Hook:** Connect corrosion, batteries, and renewable hydrogen storage.

## Learning outcomes

- Build metal-pair galvanic cells and measure voltage
- Explain galvanic corrosion and sacrificial protection
- Compare electrolysis vs. fuel cell operation (energy direction)
- Synthesize the week's experiments in a final review

## Key reactions

```
Galvanic: Zn(s) → Zn²⁺ + 2e⁻

PEM anode:  H₂ → 2H⁺ + 2e⁻
PEM cathode: ½O₂ + 2H⁺ + 2e⁻ → H₂O
Overall: H₂ + ½O₂ → H₂O
```
